** Technologies used for the assignment **

1. SpringBoot 2.6.3
2. Angular 11
3. Oracle

Notes: 
	- Tomcat is embedded in the backend and the application starts on localhost:9090
	- Angular project starts on localhost:4200
	- Query for table is provided in the misc folder
	
Tasks completed in the assignment are as follows, 
	- Create a table and add data in it
	- Expose a rest api to fetch the data
	- Display the data on UI in tabular format.
	- Add filter functionality for the table on some fields
